InstallationÔÉÅ

First make a virtual environment where you will be running astrovoightfit. (make sure you have anaconda/miniconda installed, in case you need help check this link: https://www.anaconda.com/docs/getting-started/miniconda/install#macos-linux-installation)

$ conda create -n avf python=3.11
$ conda activate avf

It's recommended to use python 3.9 or higher versions.

Now its the perfect time to install AstroVoigtFit, you can clone the repository from Github using the following command:

(avf)$ git clone https://github.com/jancami/AstroVoigtFit

(avf)$ cd path/to/astrovoigtfit

You are now all set to run AstroVoigtFit üôÇ
